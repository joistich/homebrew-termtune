# Joistich Termtune

## How do I install these formulae?

`brew install joistich/termtune/<formula>`

Or `brew tap joistich/termtune` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "joistich/termtune"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
